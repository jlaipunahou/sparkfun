    Circuit 1

  • ¶

    Turn an LED on for one second, off for one second, and repeat forever.

    In this circuit, we learned about the two main parts of an Arduino program, setup() and loop().

    In setup() the pinMode() function needed to be activated.

    void setup()
{
   pinMode(13, OUTPUT);
 }
void loop()
{
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

    Circuit 2

  • ¶

    Measure the position of a potentiometer and use it to control the blink rate of an LED. Turn the knob to make it blink faster or slower!

    In this circuit, we learned about variables and the advantages over hard-coding values. We used the potentiometer to vary the resistance in a circuit. What are everyday examples of control knobs?

    int sensorPin = 0;
int ledPin = 13;

void setup()
{
    pinMode(ledPin, OUTPUT);
}

void loop()
{
  int sensorValue;
  sensorValue = analogRead(sensorPin);
  digitalWrite(ledPin, HIGH);
  delay(sensorValue);
  digitalWrite(ledPin, LOW);
  delay(sensorValue);
}
